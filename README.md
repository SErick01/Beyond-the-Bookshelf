# Beyond-the-Bookshelf
Group Capstone Project / Fall 2025

## Description

Welcome to beyond the bookshelf! Beyond the bookshelf is a next-generation book recommendation and tracking platform. Users will be able to connect with just the click of a button! It is integrated into a cloud platform called vercel to bring easy web access to users. 

## Getting Started

Getting started is as easy as opening this link and creating  an account: https://beyond-the-bookshelf.vercel.app/index.html

## Features

* Recommendation system that elevates collaborative and content-based filtering.
* Current Reads: users can track their progress on books and rate them upon completion. (Will be available in a later version.)
* Reading Challenges: users can set yearly reading challenges.
* My Books: users can create and edit lists with their favorite books.
* User Statistics: users can view their yearly statistics
* Cat of the Day
